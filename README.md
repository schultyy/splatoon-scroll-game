# Side Scrolling Game

A simple side-scrolling game with a start screen, character selection, and game screen featuring ground collision.

## How to Play

1. Open `index.html` in your web browser to start the game.
2. Click the "Start Game" button on the start screen.
3. Select one of the three character options.
4. Control your character with these keys:
   - **Left Arrow**: Move left
   - **Right Arrow**: Move right
   - **Up Arrow** or **Space**: Jump

## Game Features

- Start screen with a title and start button
- Character selection screen with three different characters
- Side-scrolling game with ground collision
- Simple parallax background effects
- Camera follows the player

## Technical Details

The game is built using:
- HTML5
- CSS3
- JavaScript (vanilla)
- HTML5 Canvas for rendering

## File Structure

- `index.html` - Main HTML file
- `styles.css` - CSS styles for the game interface
- `game.js` - Game logic and mechanics
